## Sobre o Projeto

AutoClean é um aplicativo de ajustes de desempenho destinado ao Windows

## Contribuindo

Contribuições são sempre bem-vindas! Aqui estão as maneiras pelas quais você pode contribuir:

### Sugerindo melhorias
